# Assignment Description
This assignment focuses on the implementation and application of Stack and Queue data structures using the C programming language. The objective is to understand how stacks and queues work and how they can be used to solve real-world problems. The programs included in this assignment are:

Reverse a string using Stack
Check for Balanced Parentheses using Stack
Find Next Greater Element using Stack
Simulate a Printer Queue using Queue operations
